# 计划：示例选择器弹层（临时大网格浏览）

目标：在 `viewer/` 增加一个「示例选择器」弹层，用更大的网格视图临时覆盖主界面，便于快速挑选示例；选择后自动关闭并回到原布局。

## 验收标准
- 左侧面板提供入口按钮（例如“浏览”），可打开示例选择器弹层。
- 弹层覆盖主界面（遮罩 + 面板），包含：搜索框、关闭按钮、示例网格列表。
- 弹层默认使用网格布局（大面积展示），支持按标题/id/tag 过滤。
- 点击示例卡片会加载该 session，并自动关闭弹层；加载失败不应导致页面卡死（状态栏提示错误）。
- 弹层打开时支持 `Esc` 关闭；打开后搜索框自动获得焦点。
- 弹层与左侧列表共享“当前示例激活态”高亮（两处显示一致）。
- 不引入构建依赖，仍保持纯静态离线运行（`cd viewer && python3 -m http.server 8000`）。
